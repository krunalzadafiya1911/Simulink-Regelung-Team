%% Single Pendulum code


procedure to run the code:
1) unzip the pendulum_simulink.zip
2) add pendulum_simulink folder in the path in matlab
3) open sim_demo_pendulum.slx file
4) set the starting and final position of pendulum
5) run from the simulink
6) after the run completes, open the plotAnimataion.m file and run to see the animation output
7) here the process completes. Repeat from step 4 to rerun.
